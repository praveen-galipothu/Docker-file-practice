Here is a sample HTML file that you can use to host a simple website in a container. This file includes basic HTML structure, some CSS for styling, and a simple layout that can be served by a web server running in a container (e.g., using Nginx or Apache).

To host this website in a container, you can use a web server like Nginx. Here’s a quick guide to get started:

  1. Save the HTML file: Save the above content as index.html in a directory (e.g., my-website/).
  2. Create a Dockerfile: In the same directory, create a Dockerfile with the following content:
    dockerfile
    FROM nginx:latest
    COPY index.html /usr/share/nginx/html/index.html
    EXPOSE 80
    CMD ["nginx", "-g", "daemon off;"]
  3. Build and run the container:
    • Build the Docker image: docker build -t my-website .
    • Run the container: docker run -d -p 8080:80 my-website
  4. Access the website: Open a browser and navigate to http://localhost:8080.

This setup uses Nginx to serve the HTML file, and the website will be accessible on port 8080 of your host machine. You can modify the HTML file or add more files to the /usr/share/nginx/html/ directory in the container to expand your website.
